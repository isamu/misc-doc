# 基本的なSlashGPTの使い方


#### 起動

```
./SlashGPT.py 
```


#### ヘルプ1

```
/help
```

#### ヘルプ2

```
/help {agent_name}
```

#### サンプルの動かし方

```
/sample
````

#### 自分で質問する

```
例
```

### マニフェストの切り替え

```
/{agent_name}
```

```
/home2
```

### LLMエンジンの切り替え

```
/llm {llm_engine}
```

### 気持ちを切り替える

```
/new
```

### 履歴を見る

```
/history
```

### システムプロンプトを見る

```
/prompt
```

### マニュフェストセットを切り変え

```
/switch {dir_name}
```

### テスト！！

```
/autotest
```

### ログ

```
/verbose
```

### 音声出力
```
/audio
```

### functions
```
functions
```

### 履歴のインポート
```
/import
```
